## Customization Quick Reference

### Change Team Members
**File**: `lib/mockData.ts`
```typescript
const teamMembers = ['Your Names Here'];
```

### Change Colors
**File**: `app/globals.css` or component className
```css
--primary: oklch(your-color);
```

### Add New Metric
**File**: `components/DashboardStats.tsx`
Add a new Card component with your metric

### Add New Filter
**File**: `components/ComplaintsTable.tsx`
Add useState + filter logic

### Change Layout
**File**: Component files
Update Tailwind grid classes (md:, lg:)

## Troubleshooting

### Issue: Dashboard not loading
**Solution**: 
- Check dev server is running (`pnpm dev`)
- Clear browser cache
- Check http://localhost:3000

### Issue: Styles look wrong
**Solution**:
- Clear browser cache (Ctrl+Shift+Delete)
- Restart dev server
- Check globals.css is loaded

### Issue: Modal not opening
**Solution**:
- Click "Resolve" button on a specific row
- Check browser console for errors

### Issue: Table empty
**Solution**:
- Mock data should load automatically
- Check mockData.ts exports
- Check browser console for errors
